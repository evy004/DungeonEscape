# Dungeon Escape
Dungeon Escape is an interactive adventure game where players navigate a treacherous dungeon filled with monsters, riddles, and hidden keys. The goal is to survive deadly encounters, solve clever puzzles, and collect the keys needed to escape the dungeon.

# Features

Explore a dynamic dungeon environment with multiple rooms and secret passages.

Battle fearsome monsters using turn-based combat mechanics.

Solve riddles and puzzles to unlock doors and progress through the dungeon.

Collect keys scattered throughout the dungeon to unlock the final exit.

Experience a mix of strategy, problem-solving, and action-packed gameplay.

# How to Play

Move through the dungeon, encountering monsters and challenges.

Use your wits to solve riddles and unlock new areas.

Defeat monsters in combat to earn rewards and advance.

Find all the keys to unlock the exit and escape the dungeon.

# Getting Started

Clone the repository and run the game using your preferred environment.

Follow the in-game prompts to begin your adventure.

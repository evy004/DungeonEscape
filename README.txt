ABOUT THIS PROJECT
------------------------
This project is a Dungeon Escape adventure game where the user is able to explore a dungeon using a map to fight monsters and get keys.
The player must face a multitude of obstacles, such as fighting monsters, solving riddles, and conquering the Sorcerer!
